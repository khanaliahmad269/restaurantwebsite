# Restaurant Website

This is a full-stack web application for a restaurant, similar to McDonald's. It includes a frontend built with React and Tailwind CSS, and a backend built with Node.js and Express.

## Features

- Responsive design
- Dynamic menu display
- Location finder
- Contact form with validation
- Promotional content on homepage

## Setup Instructions

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create a `.env` file based on `.env.example` and configure your environment variables.

4. Start the server:
   ```bash
   npm start
   ```

The frontend will be available at `http://localhost:3000` and the backend API will be available at `http://localhost:5000`.
